- GitHub 仓库链接；
- README 首页截图；
- CLI 运行截图；
- 生成结果 `campaign.md` 截图；
- `shotlist.csv` 截图；
- Web Demo 截图。

- 你的 VS Code / Cursor / OpenClaw 开发环境截图；
- 真实 brief 输入和输出案例；
- 提交记录截图，证明不是临时空仓库；
- Issue / Roadmap，证明有后续开发计划；
- 一篇简短的使用报告或教程。
