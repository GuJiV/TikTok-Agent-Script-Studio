# Content Guidelines for TikTok-Style Short Videos

这个项目默认遵循以下创意原则：

## 1. 前 3 秒必须明确

每条视频要在开头直接呈现：痛点、反差、结果或好奇心。Hook 不只是一句话，也要对应一个视觉动作。

## 2. 字幕是叙事的一部分

字幕不只是复述口播，应该帮助用户快速理解节奏和重点。项目默认把每个镜头的 caption 作为结构化字段输出。

## 3. 分镜必须可拍摄

所有镜头都应该包含景别、镜头运动、画面、B-roll、声音提示和制作备注，避免只生成抽象脚本。

## 4. 不承诺不可验证结果

禁止生成“保证爆火”“7 天变现”“永久治愈”等夸大或风险表达。合规 Agent 会给出 safer rewrite。

## 5. 先生成多版本，再选择

真实短视频生产中，应至少生成 3–5 个 Hook，并保留多个剪辑版本做测试。

## 6. 原生感优先

脚本应该像平台内容，不像传统广告。尽量使用真实场景、操作过程、屏幕录制、前后对比、创作者视角。
